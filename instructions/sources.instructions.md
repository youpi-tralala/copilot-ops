# Sources de vérité

Chercher les réponses en priorité dans les sources ci-dessous.
Si la réponse n'y figure pas, le préciser explicitement avant de répondre.

## Communauté

- [Stack Overflow](https://stackoverflow.com) — privilégier les réponses les mieux notées, ignorer les réponses à vote négatif
- [Server Fault](https://serverfault.com) — idem (infrastructure, sysadmin)
- [Unix & Linux Stack Exchange](https://unix.stackexchange.com) — idem (Linux, shell)
